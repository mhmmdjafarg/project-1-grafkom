# Projek 1 Grafikom

## Stuktur untuk 1 objek
- `var object = {
          vertices: [],
          colors: [],
          mode: gl.LINES,
          shape: drawMode.LINE,
        };`
- Vertices : Berisi kumpulan titik (x,y ...)
- Colors : Berisi kumpulan rgba untuk setiap titik
- Mode : PrimitiveType webGl
- Shape : Shape identity
